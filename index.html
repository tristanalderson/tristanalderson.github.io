Cluster A

Hardware & PCB Design

Architecting the complete avionics network for the Queen's Rocket Engineering Team (QRET) 2026 hybrid rocket. Custom backplane PCB with PCIe interfaces, an ESP32 telemetry module, STM32 sensor/solenoid controller, and a CAN bus with radio forwarding for real-time ground station telemetry. Every board below was designed in KiCad — from schematic to gerbers to JLCPCB assembly.

KiCad PCB Design Cycle CAN Protocol DC-DC Conversion ESP32 STM32 I2C LoRa
Cluster B

Computer Vision & Control Systems

Built an automated rocket tracking system combining YOLOv8 computer vision with PID control to lock onto and follow a rocket through its entire flight arc. The system helped secure 2nd Place at the Launch Canada competition. Part of a 10-person launch team in the New Mexico desert, managing battery systems and critical altimeter/GPS configurations.

YOLOv8 OpenCV PID Control Python
Cluster C

FPGA & Embedded Firmware

Co-designing LiteLM — an offline, domain-specific LLM inference engine on a Xilinx Zynq UltraScale+ SoC. The ARM Cortex-A53 orchestrates while custom FPGA accelerators handle tensor computation via high-bandwidth AXI4 buses and DMA transfers for weights and KV-cache. Also developed embedded controllers at Evertz Microsystems with distributed control systems and real-time telemetry.

Zynq UltraScale+ MPSoC Vitis HLS AXI4 (Lite, Full, Stream) DMA C / C++ Verilog
